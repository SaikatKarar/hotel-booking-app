# Hotel Booking App

A full-stack hotel booking application built with the MERN Stack.

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- JWT
- Tailwind CSS

## Installation

### Clone

git clone https://github.com/SaikatKarar/hotel-booking-app.git

### Frontend

cd frontend
npm install
npm run dev

### Backend

cd backend
npm install
npm run dev
